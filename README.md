## 📋 Deskripsi Proyek

**DPM-PPTSP Kabupaten Nias** adalah aplikasi web resmi dari Dinas Pemberdayaan Masyarakat dan Pelatihan Perlindungan Tenaga Kerja Kabupaten Nias. Aplikasi ini dibuat untuk memberikan informasi publik, layanan masyarakat, dan media pengaduan bagi warga Kabupaten Nias.

Aplikasi ini dibangun menggunakan **Swift** dengan framework web **Vapor 4**, menjadikan performa aplikasi sangat cepat dan efisien.
